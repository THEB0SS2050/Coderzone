Nous llenguatges

Volem introduir nous cursos de llenguatge, el curs que estem treballant per implementare es el llenguatge de moda Reacts.

Imagina't construir blocs de Lego per crear una casa. Amb React, pots crear pàgines web de manera similar, utilitzant blocs de codi anomenats "components". Cada component té una funció específica, com ara mostrar un botó, un text o una imatge. Pots combinar fàcilment aquests components per construir pàgines web complexes i interactives.
